# seo-90-day-plan.md — Plan éditorial SEO 90 jours

Créé le 2026-07-09 à partir de l'export GSC (23/04 → 07/07/2026) et de l'audit du
corpus (146 articles). 1 article par jour, J1 = prochaine publication du daily loop.
À lire avec `seo-content-map.md` (mots-clés réservés) et `CONTENT_LEDGER.md` (inventaire).

**Le loop garde sa liberté news** : si une grosse actu tombe, elle remplace l'article
du jour et l'article planifié glisse d'un jour. Le plan est un carnet de route, pas une
prison.

## Conventions (valables pour tout le plan)

- **CTA par défaut** : note fondateur en fin d'article vers la formation IA gratuite
  (`FORMATION_FREE_URL`). Noté seulement quand un CTA différent est plus pertinent
  (Skool = communauté, page pilier, article business).
- **Schema par défaut** : BlogPosting + FAQPage (généré par `lib/seo.ts` si `faq[]`
  rempli) + BreadcrumbList. Noté seulement si HowTo ou autre est pertinent en plus.
- **Règle pilier** : chaque article lie sa page pilier de cluster (cf. content map).
- **Mise à jour outils** : tout article citant un outil précis reçoit `updatedAt`
  et une relecture à M+3 (le loop tient un registre dans le ledger).
- **Priorité** : P1 = opportunité GSC directe ou commercial fort, P2 = cluster
  stratégique, P3 = long tail / TOFU.

## Gabarits de structure par type d'article

- **Guide** : H1 promesse → H2 pourquoi / concepts clés → H2 méthode étape par étape
  (H3 par étape) → H2 pièges et erreurs → H2 FAQ → conclusion actionnable.
- **Comparatif** : H1 "X vs Y" → H2 verdict rapide (tableau) → H2 forces de X →
  H2 forces de Y → H2 cas d'usage par profil → H2 FAQ → conclusion "lequel pour toi".
- **Tutoriel** : H1 résultat visé → H2 prérequis/outils → H2 étapes numérotées (H3) →
  H2 variantes → H2 erreurs → FAQ. Schema HowTo en plus.
- **Prompt pack** : H1 → H2 comment utiliser les prompts → H2 prompts par situation
  (H3 par famille, blocs copiables) → H2 adapter/déboguer un prompt → FAQ.
- **Business/stratégie** : H1 → H2 le marché/la réalité → H2 méthode/offre → H2
  chiffres et exemples honnêtes → H2 erreurs → FAQ. CTA formation + Skool.
- **Définition/TOFU** : H1 question → réponse directe en 3 phrases (snippet) →
  H2 explication simple → H2 exemples concrets → H2 pour aller plus loin → FAQ.
